Sentiment Analyzer



The Sentiment analyzer is a sentiment analysis application, which, given a piece of text, is able to reply with the sentiment details of the text as well as the overall sentiment.

Try it by writing something below.


{% if text %}

The Sentiment Detail of "{{ text }}" is of:
{{ score[0] }} % negative
{{ score[1] }} % neutral
{{ score[2] }} % positive

{% if sentiment == 'neu' %}

The Overall Sentiment of "{{ text }}" is Neutral!

{% elif sentiment == 'neg' %}

The Overall Sentiment of "{{ text }}" is Negatif!

{% elif sentiment == 'pos' %}

The Overall Sentiment of "{{ text }}" is Positif!

{% endif %}
{% endif %}
